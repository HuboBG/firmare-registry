# Firmware Registry UI (TypeScript)

Admin UI for Firmware Registry API.

## Features
- Manage firmware types (local list) and versions (from API)
- Upload/version new firmware binaries
- Delete versions
- Manage webhooks
- 100% API-driven (no DB coupling)
- Vue 3 + Vite + TypeScript

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Env
Copy `.env.example` to `.env` and set keys.
